Countdown to WP Sevilla Meetup 2011. 

== Description ==

This widget is a countdown for <a href="http://meetup.wpsevilla.es">WP Sevilla Meetup 2011</a>. You can place it in your sidebar as any other widget.


== Installation ==

1. Upload `wp-sevilla-meetup-counter` folder to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance -> Widgets
4. Place the widget wherever you want
